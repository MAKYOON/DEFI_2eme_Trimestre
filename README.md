Pour lancer le jeu :  
-il faut que le client et le serveur soient sur le même réseau  
-le client doit lancer la console (Windows+R) puis taper "cmd" et taper "ipconfig"  
-récupérer l'adresse IPV4  
-aller dans le code client, et remplacer l'adresse IP dans la ligne suivante : "sin.sin_addr.s_addr = inet_addr("172.20.10.9");"
